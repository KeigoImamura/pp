## プライバシーポリシー  
このアプリ（Hi-Speak）は、ユーザーのプライバシーを尊重し、個人情報の保護を最優先に考えています。  

### 1. 収集する情報  
本アプリは、[収集する情報（例: メールアドレス、デバイス情報）] を収集する場合があります。  

### 2. 情報の利用目的  
収集した情報は以下の目的で使用されます。  
- アプリの改善  
- ユーザーサポート  

### 3. 情報の共有  
収集した情報を第三者と共有することはありません。  

### 4. 問い合わせ  
プライバシーに関するお問い合わせは hi.speak.hi@gmail.com までお願いします。
